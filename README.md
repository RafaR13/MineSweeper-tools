# MineSweeper-tools

MineSweeper-tools is a JavaScript-based helper tool for Minesweeper. It analyzes the current board and provides possible safe moves or mine locations based on the visible cells.

The tool includes a **Suggest Once** option to generate suggestions for the current board state, an **Auto Solve** mode that keeps updating suggestions as the game progresses, and an **Auto Play** mode that can automatically play the game.

**WARNING**: Don't use **Auto Play** if you have an account on Minesweeper.online, as you will get banned after a while. If you want to try it out, play without loggin in, and make sure you use a VPN so you don't get your IP address blocked.
